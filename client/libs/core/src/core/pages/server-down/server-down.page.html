Oops! An Error Occurred

.NET Core Server not responding

Most likely causes

  • .NET Core Windows Server Hosting bundle not installed
  • .NET Core Runtime 2.0.0 just installed (try to Reset IIS with iisreset command)
  • .json MIME types not present
